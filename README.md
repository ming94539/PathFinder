# Frontend:

To set up environment:
- cd frontend
- npm install

- cd backend
- npm install

Running Locally:
- If you want to see webapp and database connection running separately
- cd frontend
- npm start

- cd backend
- npm start

- Else, in root folder: 
- npm start

Tests:
- cd frontend
- npm test
